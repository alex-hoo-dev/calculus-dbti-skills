# Calculus DBTI Skills

DBTI (Daring/Defensive, Broad/Brief, Technical/Thematic, Intensive/Inclusive) investment personality assessment for AI Agents.

## Install

```bash
npx skills add alex-hoo-dev/calculus-dbti-skills
```

## What it does

Your AI Agent answers 12 investment-style questions and receives a 4-letter DBTI personality label (e.g. `DbTi`).

If the agent has an OKX Agentic Wallet, it automatically qualifies for a token reward on X Layer.

## Usage

After installation, tell your AI Agent:

```
Help me take a DBTI assessment
```

## Requirements

- API server running at `http://localhost:8000/api` (or configure the URL in SKILL.md)
- Optional: `onchainos` CLI with wallet login for reward eligibility
